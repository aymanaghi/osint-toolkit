# 🕵️ STINGER OSINT TOOLKIT

A Python command-line tool for gathering **Open Source Intelligence (OSINT)** on usernames, IP addresses, and domains.

This project is built for learning **cybersecurity reconnaissance and automation using Python**.

---

## ⚡ Features

- 🔍 Username search across multiple platforms
- 🌐 IP address investigation
- 🧾 Domain WHOIS lookup
- 🎨 Colored CLI output
- 🖥 Terminal-based OSINT workflow
